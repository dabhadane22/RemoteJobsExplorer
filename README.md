# RemoteJobsExplorer

A universal iOS app (iPhone + iPad) that fetches and displays remote Python jobs from the [Jobicy API](https://jobicy.com/api/v2/remote-jobs), built with a clean MVVM architecture.

## Requirements

- Xcode 15+
- iOS 17.0+ / iPadOS 17.0+

## Getting Started

1. Clone the repo
   ```bash
   git clone https://github.com/dabhadane22/RemoteJobsExplorer.git
   ```
2. Open `RemoteJobsExplorer.xcodeproj` in Xcode
3. Select an iPhone or iPad simulator
4. Press **⌘ + R** to build and run
5. Press **⌘ + U** to run all unit tests

## API

Uses the free [Jobicy Remote Jobs API](https://jobicy.com/api/v2/remote-jobs?count=20&tag=python) — no API key required.
